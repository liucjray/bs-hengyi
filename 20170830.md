## bootstrap 與 jQueryMobile 的比較
1. bootstrap 適合用於手機瀏覽器使用
2. jQueryMobile 適合用於手機 APP 使用

## jQuery 版本比較
1. jQuery-1.x.x: 支援 IE
2. jQuery-2.x.x: 不支援 IE (不建議使用)
3. jQuery-3.x.x: 2.x.x 的延伸
