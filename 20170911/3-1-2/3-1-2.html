Illustrator CC (2014)

即時形狀：矩形和圓角矩形

矩形現在有了可快速修改的圓角，包括獨立的半徑控制項。當您縮放及旋轉矩形時，圓角屬性將會保留。如今 Illustrator 會記住您的選擇，如寬度、高度、旋轉、角位樣式處理方式，因此您可以回復到原來的形狀。查看您先前描繪的路徑。在您描繪時，前一個錨點與筆尖中間會出現橡皮筋形狀的線條。

Photoshop CC (2014)

智慧型物件改良功能

自動封裝外部檔案到單一目錄，即可保留檔案的連結。您也可以將現有的內嵌智慧型物件轉換為已連結的智慧型物件。現在您可以變更圖層構圖中某一圖層的可見度、位置或外觀，然後將變更同步至所有其他圖層，從而節省時間。加上，輕鬆查看每個圖層構圖的屬性，並在智慧型物件之中切換圖層構圖。

Dreamweaver CC (2014)

CSS 設計工具改良功能

使用直覺式的視覺編輯工具，產生簡潔的標準 CSS 程式碼。快速套用文字、版面配置、色彩及其他 CSS 屬性。主要的可用性增強功能包括新的邊框工具、支援還原、工作流程的重大改良等。使用 SFTP 認證式驗證，可簡化對多部伺服器的存取，讓您安全地管理網站檔案。
